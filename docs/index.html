Motivation ðŸ¦‰

The purpose of this application is to help djs to create there own tracklists. By emulating the music selection of popular tracklists, based on the Spotify track (ML-based) descriptors, a dj can create a similar tracklist that matches the same variations in "energy" (or in general, in spotify descriptors). In this way, the new generated tracklist is similar in "energy", "feeling", "music selection", etc; but with different songs, that the dj can choose personally, therefore creating a personalized end result.

Main Functionalities ðŸ’»

Tracklist Builder is a web-based application with the purpose of enabling users to:

  • Gather data from the web by using web scraping
    • Get tracklist information from 1001tracklist about the songs that were played at a dj set
  • Analyse and visualize the data
    • Get the spotify information about the songs in the setlist
    • Graph the changes in song parameters to visualize the changes in "energy"
  • Use the data as a way to build similar tracklist
    • Use the songs stores in the database to generate a new tracklist the best-fits the parameters (graph) of the previously analyzed songs
